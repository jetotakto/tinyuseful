# Notes
# TinyUseful is a static utility website with no backend, no user accounts,
# and no stored personal data. Calculator inputs are processed locally in
# the browser and never sent to TinyUseful. The site is hosted on
# Cloudflare Pages and served only over HTTPS.
#
# If you find a security issue (e.g. XSS, broken auth on a third-party
# script, dependency vulnerability), please email the contact address
# above with a clear description and reproduction steps. We aim to
# respond within 7 days.
#
# Source code is public at https://github.com/jetotakto/tinyuseful
