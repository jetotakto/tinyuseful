JAK ČÍST: Každý sloupec je simulace mobilu 360×740. Scroll je zachován — uvidíš kolik toolů se vejde na první obrazovku a jak rychle se dostaneš k poslednímu.

A (live): baseline. Vertikální symbol 110px = každá karta zabírá ~145px výšky. Ad block uprostřed taky velký.

B (shrink): nejmenší zásah. Symbol 72px, kompaktnější padding/typo. Estetika 100% zachována. Risk: nízký.

C (2-col): radikálnější. Čtvercový symbol nahoře, 2 karty na řádek. Hustota ⇈, ale méně místa pro popis. Risk: střední (mění strukturu karty).

D (list): nejhustější, žádné symboly. Funguje jako index/menu. Estetika "directory" — hodně blízko k brutalist readeru. Risk: nejvyšší (ztrácí vizuální charakter, ale možná to chceš).

Otázka pro Tebe: Která rovnováha mezi hustotou a charakterem stránky Ti přijde správná? B = bezpečné, C = ambiciózní, D = úplně jiný směr.
